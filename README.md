# DIO_Desafio_Criativo_Quem_Nao_Vende
O projeto capacita a criação de prompts de alta performance focados em vendas. Através de etapas lógicas (Intenção, Contexto e Restrições), o usuário aprende a transformar a IA em uma aliada estratégica para gerar argumentos comerciais sólidos, contornar objeções e organizar entregas profissionais , aumentando a produtividade.
